## 📜 Core Hypothesis
Antimatter (A) generates **repulsive gravity (G-2)** through spacetime buoyancy (ρ_A < ρ_CP), while matter (M) exhibits standard attraction (G-1) via "sinking" (ρ_M > ρ_CP).
